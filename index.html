HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • !DOCTYPE html: tells the browser what document type it is
  • lang: tells the browser the language of the webpage
  • style: defines CSS styles associated with a website
  • base: defines the base URL for a webpage
  • link: connects an external resource
  • meta: defines metadata such as description, character set, keywords, etc
  • charset: sets our character encoding
  • div: create a division in the page
  • span: create an inline division
  • p: represents a paragraph or block of text
  • ul,ol,li: elements that represent list items
  • br: linebreak
  • anchor a: creates links to a webpage
  • href: attribute points to the URL for the link
  • footer: appears at the bottom of the page

CSS

  • box-shadow: adds shadow effects around an element's frame
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • card: designates this as a classs selector

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • cd: allows us to change directories
  • cd . .: used to navigate backwards
  • pwd (print working directory): print out the directory that we are currently in to the terminal
  • mkdir: create a directory
  • ls: see a list of files
  • touch: create a new file
  • git config: change the name associated with my Git commits
  • git revert: revert to make a fix and complete a command
  • feature branch: a new branch to add something new to our projects
  • pull request or PR: used to merge the feature branch into the base branch
  • main: default branch. Note: we shouldn't code direcly on this branch
  • add -A: add code to the current working branch

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • console.log: method to output a message to the web console by adding an argument
  • if: introduces decision-making into our code
  • else if: used when you have more than one condition
  • loop: uses the predictable pattern of indices to perform a task on all the items in an array
  • x++: increase by one every time the code loops
